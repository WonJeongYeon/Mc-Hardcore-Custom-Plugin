# MC Collector 플러그인

이 프로젝트는 마인크래프트 서버의 하드코어 난이도 경험을 커스터마이징하기 위해 개발 중인 `mc_collector` 플러그인입니다.

## 프로젝트 개요

`mc_collector` 플러그인은 마인크래프트 서버 플레이어들에게 보다 독특하고 도전적인 하드코어 난이도 경험을 제공하는 것을 목표로 합니다. 궁극적으로는 하드코어 난이도 클리어 방식을 다양하게 커스텀할 수 있는 기능을 제공할 예정입니다.

현재 개발 초기 단계이며, 플러그인의 기반을 다지고 기본적인 서버 운영에 필요한 기능들을 구현하는 데 집중하고 있습니다.

## 현재 개발 상태

*   **기본적인 서버 운영 기능 구현:** 플러그인의 기본적인 구조와 서버에서 정상적으로 작동하기 위한 최소한의 기능들이 구현되어 있습니다. (예: 플러그인 로드/언로드, 기본 명령어 처리 등)
*   **하드코어 난이도 커스터마이징 기능은 아직 구현되지 않음:** 최종 목표인 하드코어 난이도 관련 커스텀 기능은 현재 개발 계획 단계에 있으며, 향후 업데이트를 통해 추가될 예정입니다.

## 향후 개발 계획

*   **하드코어 난이도 클리어 조건 커스터마이징:** 일반적인 엔더 드래곤 처치 외에, 새로운 클리어 조건(예: 특정 퀘스트 완료, 모든 발전 과제 달성 등)을 추가하는 기능.
*   **난이도 동적 조절 시스템:** 플레이어의 행동이나 서버 상황에 따라 하드코어 난이도의 특정 요소를 동적으로 조절하는 기능.
*   **커스텀 이벤트 및 챌린지:** 하드코어 경험을 더욱 흥미롭게 만들 사용자 정의 이벤트나 챌린지 추가.

## 시작하기 (개발자 및 서버 관리자용)

### 전제 조건

*   Java Development Kit (JDK) 17 이상
*   Apache Maven
*   Bukkit, Spigot, Paper 마인크래프트 서버 소프트웨어

### 플러그인 빌드 방법

1.  GitHub에서 이 저장소를 로컬로 클론합니다:
    ```bash
    git clone https://github.com/your-username/mc_collector.git
    cd mc_collector
    ```
2.  프로젝트 루트 디렉토리에서 Maven을 사용하여 플러그인을 빌드합니다:
    ```bash
    mvn clean package
    ```
    이 명령을 실행하면 `target/` 디렉토리에 `mc_collector-1.0.1.jar` 형태의 실행 가능한 JAR 파일이 생성됩니다.

### 서버에 플러그인 설치

1.  빌드된 `mc_collector-1.0.1.jar` 파일을 마인크래프트 서버의 `plugins/` 폴더에 업로드합니다.
2.  마인크래프트 서버를 시작하거나 재시작합니다.
3.  서버 콘솔에 `plugins` 명령어를 입력하여 `MC Collector` 플러그인이 정상적으로 로드되었는지 확인합니다.

## 사용법

현재는 기본적인 서버 운영 기능만 포함하고 있으므로, 특별한 사용자 설정이나 명령어는 없습니다. 향후 하드코어 난이도 커스터마이징 기능이 추가되면 해당 기능에 대한 자세한 사용법과 설정 방법이 업데이트될 예정입니다.
